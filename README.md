# ☄ Cosmos Client — Fabric Mod для Minecraft 1.21.1

Утилитный мод для **одиночной игры** с красивым космическим GUI.

---

## 📦 Как собрать .jar и установить

### Требования
- **Java 21+** — [Скачать Adoptium](https://adoptium.net/)
- **Gradle 8.8** — устанавливается автоматически

### Шаги

```bash
# 1. Зайди в папку проекта
cd CosmosClient

# 2. Собери мод
gradle build

# 3. Готовый .jar будет в:
#    build/libs/CosmosClient-1.0.0.jar
```

### Установка
1. Установи [Fabric Loader 0.16.5](https://fabricmc.net/use/) для Minecraft **1.21.1**
2. Установи [Fabric API 0.102.0+1.21.1](https://modrinth.com/mod/fabric-api)
3. Скопируй `CosmosClient-1.0.0.jar` в папку `.minecraft/mods/`
4. Запусти Minecraft с профилем **Fabric 1.21.1**

---

## 🎮 Управление

| Кнопка | Действие |
|--------|----------|
| `RIGHT SHIFT` | Открыть / закрыть GUI |
| `Z` (удерживай) | Зум (когда Zoom включён) |
| `Scroll` во время зума | Регулировка приближения |
| `ESC` | Закрыть GUI |

## ✨ Список читов

### ⚔ Combat
| Чит | Описание |
|-----|----------|
| **KillAura** | Авто-атака ближайших существ в радиусе 4.5 блока |
| **Strength** | Усиление урона (эффект Силы V) |
| **AntiKnockback** | Полная блокировка отброса при ударе (через Mixin) |

### 🚀 Movement
| Чит | Описание |
|-----|----------|
| **Flight** | Свободный полёт в выживании |
| **Speed** | Скорость передвижения (Speed V) |
| **NoFall** | Нет урона от падения |
| **HighJump** | Высокий прыжок (Jump Boost V) |
| **SprintAlways** | Постоянный спринт |
| **NoSlowdown** | Нет замедления при еде/луке/щите |

### 🧑 Player
| Чит | Описание |
|-----|----------|
| **Regeneration** | Быстрое восстановление HP (Regen V) |
| **AntiHunger** | Голод не расходуется (Saturation) |
| **AutoEat** | Автоматически ест при низком голоде |
| **WaterBreathing** | Дыхание под водой + скорость дельфина |
| **NoFire** | Иммунитет к огню и лаве |
| **AutoTool** | Автоматически выбирает лучший инструмент |

### 🌍 World
| Чит | Описание |
|-----|----------|
| **FastMine** | Быстрая добыча (Haste V) |
| **NoWeather** | Убирает дождь и гром |
| **XRay** | Night Vision для лучшей видимости под землёй |

### 👁 Render
| Чит | Описание |
|-----|----------|
| **FullBright** | Полная яркость в темноте (Night Vision без эффекта) |
| **EntityESP** | Подсветка мобов и игроков через стены (Glowing) |
| **Zoom** | Приближение как в OptiFine (удерживай Z) |

## 🎨 GUI

- Открывается через **RIGHT SHIFT**
- Тёмный космический фон со звёздами и туманностями
- Радужные анимации на активных читах
- Категории слева, карточки с читами справа
- Нажми на карточку — включить/выключить чит
- HUD в углу экрана показывает активные читы

---

## ⚠ Примечания
- Мод предназначен только для **одиночной игры**
- Не используй на серверах — это нарушает правила
- Fabric API обязателен
